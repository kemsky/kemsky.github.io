public function compare(a:Object, b:Object, options:uint = 0, equals:Boolean = false):int

Parameters

a:Object — first item
 
b:Object — second item
 
options:uint (default = 0) — combination of Stream.CASEINSENSITIVE | Stream.DESCENDING | Stream.NUMERIC
 
equals:Boolean (default = false) — compare is used to check equality

Returns
int — -1 if a < b, 0 if a == b and 1 if a > b
